# Introduction #

This chapter explain you how to create the Arduino\_to\_XMC1100 for XMC1100 via DAvE.

You need:
  * The board: Infineon XMC2GO Board
  * Environment: DAVE™ – Free Development Platform for Code Generation (DAVE™ 3.1.8 or later)
